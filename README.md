# trade management

### CRM сервис для управления продажми

### Стек технологий:
+ Java 17
+ PostgresSql 14
+ Maven
+ SpringBoot
+ lombok 1.18.22

### Требования к окружению:
+ Java 17
+ Maven 3.x
+ PostgresSql 14

### Запуск приложения
+ Создать базу данных

```create database trade_management;```

+ Запуск приложения с maven. Перейдите в корень проекта через командную строку и выполните команды:

``` mvn clean install```
``` mvn spring-boot:run```
